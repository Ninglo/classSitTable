# 在线班级座位表（纯前端）

一个给老师使用的多班级座位轮换工具，数据仅保存在本地浏览器。

## 已实现核心能力

- 多班级独立管理（每个班独立名单、布局、课程信息、备注）
- 名单导入（粘贴文本 / CSV / TXT）
  - 自动去重，最多 36 人
- 两种布局模式
  - `小组布局`：自动 3~6 组，每组最多 6 人
  - `三大横排`：3 排座位，按规则轮换
- 一键轮换
  - 小组布局：最后一组到第一组，组内 3 排轮换（左右不换）
  - 三大横排：`3->1, 1->2, 2->3`，排内执行两边向中间位移
- 课堂信息编辑
  - 周次、上课时间、校区、楼栋、教室、侧边备注
- 打印分享版
  - 支持直接打印 / 导出 PDF 发给家长和学生
- 本地备份
  - 导入/导出 JSON

## 运行项目

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 构建预览

```bash
npm run build
npm run preview
```

## 数据说明

- 数据仅在当前浏览器保存（LocalStorage）
- 清理浏览器缓存可能导致数据丢失
- 建议定期导出 JSON 备份
